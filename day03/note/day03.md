### 面向对象编程 

面向对象编程是一种新的编程方式，它是一种将一些零散变量与函数封装在一起统一解决问题的编程方式。

**类**  （面向对象编程思想中的**自定义类型**） 

```
class Person {
    name: string;
    age: number;
    gender: string;
    married: boolean;
}
```

**对象：**（通过类型的定义，创建出来的一个具体的实例）

```
let p1 = new Person('zs', 15, '男', false)
let p2 = new Person('ls', 25, '女', true)
new Date()
new Array()
new Option()
new AMap.Map(document.getElementById('contaniner'))
new AMap.Map()
```
