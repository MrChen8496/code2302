# Vue2Project DAY06

#### 实现新增电影业务

**业务需求：**点击新增电影菜单，看到一个表单，填写后点击提交，新增成功。

**实现步骤：**

1. 准备好一个静态页面，包含一个大表单， 9个表单项。
2. 针对每一个表单项处理收集的数据。
3. 点击提交按钮，实现新增业务。
4. 添加表单验证。
